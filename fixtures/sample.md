# Onboarding-Leitfaden

## Erster Arbeitstag

Neue Mitarbeitende erhalten am **ersten Arbeitstag** ihren Laptop und die Zugangskarte im Empfang. Die IT-Einrichtung dauert etwa zwei Stunden.

## Zugriffe beantragen

Zugriffe auf interne Systeme werden über das [Self-Service-Portal](https://portal.example.com) beantragt und von der Teamleitung freigegeben.

```bash
# Beispiel: VPN-Profil installieren
vpnctl install --profile firma-standard
```

Nach der Installation ist das Firmennetz aus dem Homeoffice erreichbar.
